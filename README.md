# SpeedCloud-UI

使用Vuetify框架，界面模仿百度网盘Web端

## 程序启动
```
npm install
npm run serve
```
## 文件说明
* nginx.conf：nginx网关配置
* speedcloud.sql：后端数据库SQL生成代码
